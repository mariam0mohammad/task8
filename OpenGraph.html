Open Graph (OG)

What is Open Graph (OG)?

Open Graph (OG) is a protocol developed by Facebook that enables web pages to become rich objects in a social graph. It allows developers to control how their content is displayed when shared on social media platforms like Facebook, LinkedIn, Twitter, and others. By adding specific metadata to a webpageâ€™s section, you can define how the page looks when someone shares its link.

How to Use Open Graph?

Add OG meta tags to the section of your HTML document. These tags provide metadata about the webpage, such as the title, description, image, and URL.
